# 工作日誌與紀錄 (ccwLog)

本目錄 (`ccwLog/`) 用於存放開發過程中的工作日誌、對話紀錄以及各種臨時性的設計筆記。

## 📂 內容說明

- **工作日誌 (`.md` 檔案)**：紀錄特定日期的開發進度、需求討論、以及 AI 代理協作的指令與反饋（例如 `0615-readmeNotherDocs.md`）。
- **決策紀錄**：在此目錄下的文件可作為開發決策的備忘錄，追蹤為何建立特定的規格或修改特定的程式碼。

> **提示**：這些文件主要是給開發者與 AI 助手作為上下文參考，並不屬於專案的正式發佈規格（SSoT）。正式的領域模型與系統意圖請參閱各專案下的 `docs/ssot/` 目錄。
