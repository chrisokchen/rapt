## SOP

### 步驟 0：ASSERT payload 完整性

```
ASSERT: payload.target_path 以 .dbml 結尾
ASSERT: payload.content.tables 存在且非空
ASSERT: payload.source_evidence 存在
ASSERT: payload.dsl_version == "3.3.0"
若失敗 → EMIT 錯誤，停止
```

### 步驟 1：LOAD format rules

LOAD REF [rapt-form-dbml::references/dbml-format-anchor.md]

### 步驟 2：RENDER DBML content

依 payload.content.tables 和 payload.content.refs 渲染 DBML：

```
// Generated by rapt-form-dbml
// Source: {source_evidence[0].ref}
// DSL: annotated DBML v3.3.0

{TableGroup blocks if present}

{Table blocks with v3.3 annotations}

{Ref declarations}
```

### 步驟 3：VALIDATE 反模式

對照 `rapt-modeling::rules/dbml-v33-annotation-rules.md` AP-01~AP-05：

若有違規：EMIT 錯誤給 caller（不自行修正），停止。

### 步驟 4：WRITE `payload.target_path`

### 步驟 5：EMIT 完成

```
✅ rapt-form-dbml: 已寫入 {target_path}（{N} 個 Table）
```
