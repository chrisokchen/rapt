## 1. Proposal Summary

- **標題**：新增商品評論
- **來源**：`what-if/0619-addComment.md`
- **摘要**：允許顧客對「已購買」的商品撰寫評論（1~5 星＋文字），評論與訂單綁定；商家管理自店評論（顯示/隱藏/刪除），平台 admin 管理全平台評論。
- **entities**：ProductReview（新）、Product、Order、OrderItem、Member
- **actors**：consumer、merchant、platform-admin
- **明示約束**：僅已購買者可評論（評論→訂單→訂單項目→商品）；狀態＝顯示/隱藏/刪除
- **本身模糊處**：資格認定層級、刪除語意、評分聚合、下架保留、稽核範圍皆未定義

## 7. Data / API / UI / Permission Impact

- **Data**：新 aggregate ProductReview（評分/內容/狀態/時間/評論者/評論對象/訂單綁定）；評分聚合是否反正規化；對 Order/OrderItem/Product/Member 外鍵與資格查詢。
- **API**：新 product-review API（create/list/read + moderate）；建立時資格驗證；既有 product API 是否暴露評分摘要。
- **UI**：顧客評論表單、商品頁評論列表（須補商品詳情頁）、商家/admin 評論管理列表。
- **Permission**（三種既有模式可直接套用，**零新模式**）：
  - consumer `review_create_own` — 比照 `wishlist_*_own`（memberId 條件）
  - merchant `review_list_store` / `review_moderate_store` — 比照 `*_update_store`（store 隔離）
  - platform-admin `review_list_all` / `review_moderate_all` — 比照 `audit_log_read_all`
  - 待決：商家可否任意隱藏（言論/審查邊界）。
