## 萃取的關鍵業務概念

1. SmallBiz 是服務中小零售商的 B2C 電商 SaaS，商家透過月租方式使用平台。
2. 庫存準確與避免付款後缺貨是商家與消費者共同的最高優先問題。
3. 促銷、優惠券、會員折扣與點數折抵需要明確且可預測的計算規則。
4. 消費端體驗必須行動優先、快速、欄位少、付款安全並可追蹤訂單。
5. 商家後台必須極簡，支援商品、訂單、庫存、出貨、報表與促銷管理。
6. 第一版需涵蓋付款、物流追蹤、電子發票或稅金計算、退貨申請與會員忠誠機制。
